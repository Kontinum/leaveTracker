## About Leave Tracker
Simple Laraver API with sanctum authorization. This project was a part of assignment for backend developer position.

Three types of users: admin manages other users(create, edit, delete) and teams, regular users makes request for leave inside teams(no overlaps and max available days limitation) and managers approves/rejects leave request inside teams.
